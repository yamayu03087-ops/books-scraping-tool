# books-scraping-tool

Pythonで商品データ（タイトル・価格）をWebサイトから取得し、整形・整理してCSV形式で出力するスクレイピングツールです。

---

## ■ 対象サイト
楽天市場（コスメ・ヘアケアカテゴリの商品一覧ページ）

---

## ■ 概要

本ツールは、Webページから商品情報を自動取得し、以下の処理を行うことで実用的なデータとして整理します。

- 商品タイトルの取得
- 価格データの抽出と数値化
- 重複データの削除
- 価格順でのソート
- CSVファイルへの出力

---

## ■ 使用技術

- Python
- requests
- BeautifulSoup
- pandas

---

## ■ 機能詳細

### 1. スクレイピング
楽天市場の商品一覧ページから商品名と価格を取得

### 2. データ整形
- 価格から「¥」「,」を除去し数値化
- 欠損データ（None）の除外処理

### 3. データ整理
- 商品タイトルを基準に重複削除
- 価格の昇順ソート

### 4. CSV出力
整形済みデータを `books.csv` に保存

---

## ■ 工夫した点

- HTML構造の変化を考慮した安全なデータ取得（Noneチェック）
- 取得失敗時でも処理が止まらないエラーハンドリング
- 実務を想定した「取得→整形→出力」の一連処理の自動化
- pandasによる効率的なデータ処理
